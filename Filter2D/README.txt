To run the code :

1. Open the Filter2D.xise 'Xilinix ISE project file'.
2. Click on Simulation radio button.
3. Go to 'readWriteCache_tb' and right click and select 'Set as Top Module'.
4. Double click on 'readWriteCache_tb' to see the code in editor.
5. Go to lines 227-237, to change the kernel values.
6. Run the 'Simulate Behavioral Model' and the timing diagram is popped up and run for atleast 165 microSec to get the result.  
7. Open Matlab to see the result.

To see MATLAB result :

1. Change the present working directory (pwd) of MATLAB to current folder.
2. Run the 'displayImage.m' to see the result.
